# Asset and dataset licensing checklist

Before making this repository public, record the source and license for:

- `overlay/data/assets/`: Kangaroo visual meshes;
- `overlay/protomotions/data/assets/Kangaroo/kangaroo_grippers_ias.xml`;
- the generated `kangaroo_grippers_ias_configured.usd`;
- any SOMA23/BONES source motion or derived keypoint file;
- rendered media if the source asset license imposes attribution requirements.

The generated USD is derived from the XML and meshes and therefore does not
automatically acquire a new permissive license. Do not add a blanket software
license to third-party robot assets until their original terms are known.
